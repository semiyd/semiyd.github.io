内核启动后升级或者读出kernel办法

升级：

假设uImage的分区是mtd7
则：
flash_erase /dev/mtd7 0 0
nandwrite -p /dev/mtd7 -p /boot/uImage
同理可以更新u-boot等分区。
但是需注意。kernel写flash，默认使用的ECC方式，必须和读取u-boot或者uImage的方式相同。
因为有时候u-boot和uImage的ecc方式是不同的。

读取：

dd if=/dev/mtd0 of=boot bs=12M count=1
