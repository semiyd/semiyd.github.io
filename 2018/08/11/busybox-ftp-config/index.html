busybox ftp config

ftpd配置根目录可写

根据这个文章，其实配置都是放在/etc/inetd.conf里面的。
包括之前的telnet服务。
ftp stream tcp nowait root /usr/sbin/ftpd /usr/sbin/ftpd
telnet stream tcp nowait root /usr/sbin/telnetd /usr/sbin/telnetd -i
所以之前在rcS里面放的telnetd启动，其实可以去掉了。
所以在这里配置的服务，ps是看不见tftpd的进程的
在ubuntu下ftp 192.168.1.5可以登录
在inetd.conf的ftpd最后配置 -w参数，让他可上传。ftp stream tcp nowait root /usr/sbin/ftpd /usr/sbin/ftpd -w
这时候用linux的命令行下使用ftp来上传下载开发板上的文件
put /root/Desktop/hfdev.c /usr/hfdev.c
get hf_user /root/Desktop/hf_user
都是可以的
