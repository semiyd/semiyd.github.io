Cross Compile OpenSSH

Install arm cross compiler - (eg. what is arm-linux-gcc and how to install this in ubuntu)
Download:
● Zlib
● OpenSSL
● OpenSSH

Build Zlib:
cd zlib-1.2.7
./configure –prefix=$PWD/../../rootfs_ndc
修改Makefile：
CC=arm-linux-gcc
AR=arm-linux-ar
CPP=arm-linux-gcc -E
LDSHARED=arm-linux-gcc

Build OpenSSL:
./Configure dist –prefix=$PWD/../../rootfs_ndc
make CC=”arm-linux-gcc” AR=”arm-linux-ar r” RANLIB=”arm-linux-ranlib”
make install

Build OpenSSH:
./configure –host=arm-linux –with-libs –with-zlib=$PWD/../../rootfs_ndc –with-ssl-dir=$PWD/../../rootfs_ndc –disable-etc-default-login CC=arm-linux-gcc AR=arm-linux-ar
make
修改Makefile：
STRIP_OPT=–strip-program=arm-linux-strip
install: $(CONFIGFILES) $(MANPAGES) $(TARGETS) install-files install-sysconf （多余的两个install项删掉）
make install DESTDIR=$PWD/../../rootfs_ndc

Cross Compile libssh

libssh是应用程序要用到ssh的时候所依赖的库。

从https://www.libssh.org/get-it/下载libssh-0.7.5

加压，然后
mkdir libssh-build
cd libssh-0.7.5
mkdir build
拷贝toolchain.cmake到libssh-0.7.5/build目录
cd libssh-0.7.5/build
cmake -DCMAKE_TOOLCHAIN_FILE=./toolchain.cmake ..
make
make install DESTDIR=$PWD/../../libssh-build

编译目录：（这个代码要求编译和源码不同路径）
libssh-0.7.5/build
安装目录
libssh-build/
