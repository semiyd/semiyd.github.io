Python 2.7 and Pyzmq cross-compile

Python 2.7

tar -xf Python-2.7.13.tar.xz
tar zxvf Python-2.7.13-xcompile.patch.tar.gz
mkdir python2_7_13_for_x86_64
mkdir python2_7_13_for_arm
mv Python-2.7.13-xcompile.patch python2_7_13_for_arm/

cd python2_7_13_for_x86_64
../Python-2.7.13/configure –prefix=pwd
make -j8
make install

cd ../Python-2.7.13/
patch -p1 < ../python2_7_13_for_arm/Python-2.7.13-xcompile.patch
cd ../python2_7_13_for_arm
mkdir result
修改/Python-2.7.13/setup.py，找到下面这两行，注掉：
3 # If a module has already been built statically,
4# don’t build it here
5- ifext.name insys.builtin_module_names:
6- self.extensions.remove(ext)
7+ #ifext.name in sys.builtin_module_names:
8+ # self.extensions.remove(ext)
然后继续：

../Python-2.7.13/configure –host=arm-none-linux-gnueabi –build=i586-linux-gnu –enable-ipv6 –enable-shared ac_cv_file__dev_ptmx=”yes” ac_cv_file__dev_ptc=”no”

make HOSTPYTHON=../python2_7_13_for_x86_64/python HOSTPGEN=../python2_7_13_for_x86_64/Parser/pgen BLDSHARED=”arm-none-linux-gnueabi-gcc -shared” CROSS_COMPILE=arm-none-linux-gnueabi- CROSS_COMPILE_TARGET=yes HOSTARCH=arm-none-linux-gnueabi BUILDARCH=i586-linux-gnu -j8

make install HOSTPYTHON=../python2_7_13_for_x86_64/python BLDSHARED=”arm-none-linux-gnueabi-gcc -shared” CROSS_COMPILE=arm-none-linux-gnueabi- CROSS_COMPILE_TARGET=yes DESTDIR=$PWD/result/

pyzmq

从https://github.com/zeromq/pyzmq下载pyzmq-master.zip

am335x板子上安装：
apt-get install cython
export PATH=/home/mgc/embedded/codebench/bin:/usr/local/sbin:/usr/local/bin:/usr/bin:/bin:/usr/lib/jvm/java-8-oracle/bin:/usr/lib/jvm/java-8-oracle/db/bin:/usr/lib/jvm/java-8-oracle/jre/bin
export MGLS_LICENSE_FILE=/home/mgc/license.lic
export CC=”arm-linux-gcc”
export LDSHARED=”arm-linux-gcc -shared”
临时把/home/share/libzmq/arm-linux-4.9.1/lib/下面的动态库拷入/home/share/python/2.7.13/Board/arm-none-linux-gnueabi-4.9.1/usr/local/lib/（因为后续会硬是找不到-lzmq的）
创建setup.cfg，内容如下：
[global]
zmq_prefix = /home/share/libzmq/arm-linux-4.9.1/
have_sys_un_h = False

[build_ext]
libraries = python2.7
library_dirs = /home/share/python/2.7.13/Board/arm-none-linux-gnueabi-4.9.1/usr/local/lib
include_dirs = /home/share/python/2.7.13/Board/arm-none-linux-gnueabi-4.9.1/usr/local/include/python2.7

[bdist_egg]
plat-name = linux-armv
执行：
python2.7 setup.py build –zmq=/home/share/libzmq/arm-linux-4.9.1/
/home/share/python/pyzmq/pyzmq-master/build/lib.linux-i686-2.7/里面的zmq/文件夹，拷入/home/share/python/2.7.13/Board/arm-none-linux-gnueabi-4.9.1/usr/local/lib/python2.7/
测试方法：
[root@Honeywell ~]# python2.7
Python 2.7.13 (default, Jan 25 2018, 09:33:56)
[GCC 4.9.1] on linux2
Type “help”, “copyright”, “credits” or “license” for more information.
>>> import zmq
>>> print(“libzmq version is %s” % zmq.zmq_version())
libzmq version is 4.2.2
>>> print(“pyzmq version is %s” % zmq.__version__)
pyzmq version is 17.0.0
