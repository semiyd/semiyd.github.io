ntfs-3g cross-compile

默认busybox并不支持ntfs的u盘的挂载
ntfs-3g就是支持这个功能的开源软件

从https://www.tuxera.com/community/open-source-ntfs-3g/下载ntfs-3g_ntfsprogs-2017.3.23.gz
用鼠标解压压缩包。安装目录定为/home/xxx/NDC/06.00.00.00/arm-none-linux-gnueabi-4.9.1/ntfs-3g/ntfs_build/
./configure CC=arm-linux-gcc –host=arm-linux –disable-ldconfig –with-fuse=internal
make -j8
mkdir /home/xxx/NDC/06.00.00.00/arm-none-linux-gnueabi-4.9.1/ntfs-3g/ntfs_build/lib
make install DESTDIR=/home/xxx/NDC/06.00.00.00/arm-none-linux-gnueabi-4.9.1/ntfs-3g/ntfs_build

内核要勾上<*> FUSE (Filesystem in Userspace) support，这个是fuse驱动。
fuse其实分两部分，一个是内核驱动，一个是用户空间的程序，也就是现在ntfs-3g已经编进去的部分。

插入u盘
mount.ntfs-3g /dev/sda1 /mnt
卸载u盘
umount /mnt

注：
如果mount.ntfs-3g遇到ntfs-3g-mount: fuse device is missing, try ‘modprobe fuse’ as root
其实是因为内核要勾上<*> FUSE (Filesystem in Userspace) support
