然后，通过命令：
mknod /dev/globalmem c 230 0 创建设备节点
然后echo “hello” > /dev/globalmem
然后cat /dev/globalmem，就会得到刚刚的hello

cat和read的异同点
cat: 会一直read，直到return 0为止。所以cat的本意是，读一个file，一直读，读到没有东西可以读为止。也就是一下子全部读出来。
