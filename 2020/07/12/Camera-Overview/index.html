Camera Overview

参考官方的《overiew.pdf》

最上层：（Camera API v2）
1.android.hardware.camera2的包.
包括：
android.hardware.camera2.CameraDevice
android.hardware.camera2.CameraManager
2.是AIDL界面（Camera API v2这一侧），用于和中间的Native层的CameraService进行RPC通信的。位于frameworks/av/camera/aidl/android/hardware
包括：:
android.hardware.camera2.ICameraDeviceUser //the interface to a specific opened camera device
android.hardware.camera2.ICameraService //the interface to the camera service
例如：frameworks/av/camera/aidl / android / hardware / ICameraService.aidl，对应android.hardware.camera2.ICameraService

中间层：（CameraService）
这一层其实是Native framework，原生框架，c++编写。位于frameworks/av/
与framework层中的android.hardware.camera2.CameraDevice和android.hardware.camera2.CameraCaptureSession类相对应。
1.包括aidl中CameraService中间层这一侧的Binder IPC interface，用于和Camera API v2通信。
具体包含了ICameraServiceListener and ICameraDeviceCallbacks两个给Camera API v2的回调函数。
2.真正的CameraService位于frameworks/av/services/camera/libcameraservice/CameraService.cpp，会和HAL交互

下层（HAL/HIDL）
要实现三个界面：
ICameraProvider: For enumerating individual devices and managing their status.
ICameraDevice: The camera device interface.
ICameraDeviceSession: The active camera device session interface.
以上三个接口，样例代码是：
CameraDevice.cpp.pdf
CameraDeviceSession.cpp.pdf
CameraProvider.cpp.pdf
